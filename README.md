# cub3d_map_generator

## Install
```
git clone
python3 map_generator.py
```

- You can clone this generator in your cub3d repository
- Add your texture path (ex : ./textures/)
- Add your RGB ceiling datas (ex : 220,40,0)
- Same for RGB floor datas
- The map.cub file is created !
